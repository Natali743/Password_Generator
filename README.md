## Описание программы

GUI-приложение «Random Password Generator» с возможностью настройки параметров пароля, сохранением истории.
---

## Требования для запуска

Что нужно установить на компьютер:

- Python 3.10 или выше
- Библиотеки: `pip install Tkinter

---

## Как запустить

https://github.com/Natali743/Password_Generator.git
________________________________________
Как пользоваться
При запуске появляется окно, которое позволяет установить длину пароля с помощью ползунка, а также сделать выбор состава пароля: цифры, буквы или спецсимволы.
По кнопке "Сгенерировать", в поле формируется пароль, а также отображается в таблице истории.
Есть возможность очистки истории.
